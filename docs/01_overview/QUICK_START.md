# QUICK START - 3分钟快速启动

1. 解压项目包到任意目录。
2. 进入项目根目录：

```bash
cd dl-waf-hybrid
```

3. 安装依赖：

```bash
pip install -r requirements.txt
```

4. 运行启动脚本（Windows / Linux）：

```bash
# Windows
start-waf.bat
# Linux/macOS
bash start-waf.sh
```

5. 启动 WAF 服务：

```bash
python main.py
```

6. 打开 Web 管理界面：

http://localhost:8082
